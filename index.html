主题编号
给大括号思维导图的各级子主题自动编号（根/中心主题不编号），编号显示在卡片左上角。
在各种结构里，把某个节点直接拖开或拖动连线接到它的小圆点手柄，即可单独微调它到父级的连线段长（阅读模式下手柄不显示）；也可右键该节点用滑条精调。整图统一间距用顶部「整理」里的滑条。
默认快捷键集中在键盘左半部分，方便右手用鼠标、左手按键。点右侧按键即可录制新快捷键，录制时按 Esc 取消、按 退格/Del 清空该项；撞键会自动让出。
关闭"新建框"后，大括号只画一个开口，把已有文本框拖到开口处即可吸附成父级/目标；
按住 Ctrl + 左键可把括号内/反向目标的文本框拖出来。
整理时会为每条连线自动选用朝向对方的锚点；勾选「自动新建」后，已有锚点都不合适时会自动加一个锚点。勾选「禁止共用」后，落在同一文本框同一点的多条连线会被自动错开到各自独立的锚点。以上仅作用于蜘蛛网思维导图，与大括号思维导图的设置相互独立。

AI 设置
接入你自己的本地模型或 API Key · 只存本地，绝不上传
本地模型（Ollama / LM Studio / llama.cpp 等）与各家云服务都用这一套 OpenAI 兼容接口，填对地址即可。
填错会报 invalid model name。点上面「列出可用模型」从你的服务里直接选；或本地跑 ollama list 看名字（如 qwen2.5:7b），云服务填官方模型名（如 deepseek-chat）。
密钥只保存在你本机，随请求直接发往你填写的地址，本软件不经过任何中转服务器。
